Contexto

O que e Remotion e Claude Code?

Framework

Remotion

Framework open-source que permite criar videos programaticamente usando React. Cada frame do video e um componente. O resultado final e um MP4.

Ferramenta AI

Claude Code

CLI da Anthropic que permite escrever e editar codigo usando linguagem natural direto no terminal. Voce descreve, ele implementa.

Quando voce combina os dois, o fluxo fica assim: voce descreve o video em texto, o Claude escreve os componentes React, e o Remotion renderiza em MP4.

Conceito chave

O que sao Agent Skills?

Agent Skills sao conjuntos de instrucoes curadas que ensinam agentes de IA (como Claude Code) a trabalhar corretamente dentro de projetos especificos. No caso do Remotion, os skills ensinam o Claude a:

  • Criar componentes React no padrao do Remotion
  • Usar hooks como useCurrentFrame() e useVideoConfig()
  • Registrar compositions corretamente
  • Aplicar animacoes frame a frame
  • Seguir boas praticas de performance
i

Sem os skills, o Claude consegue criar videos com Remotion — mas com eles, o resultado e significativamente melhor porque ele conhece as convencoes e padroes do framework.

O skill oficial do Remotion ficou no top do marketplace skills.sh com mais de 25.000 instalacoes nos primeiros dias de lancamento (janeiro 2026).

Mao na massa

Setup passo a passo

1

Crie o projeto Remotion

Abra o terminal e rode o comando abaixo. Ele cria um projeto completo com tudo configurado:

npx create-video@latest

Quando perguntar, escolha:

  • Template: Blank
  • TailwindCSS: Sim
  • Install Skills: Sim
2

Instale as dependencias

Entre na pasta do projeto e instale:

# Substitua "meu-video" pelo nome que escolheu
cd meu-video
npm install
3

Rode o preview

Isso abre o Remotion Studio no browser — um player interativo onde voce ve o video em tempo real:

npm run dev

Acesse http://localhost:3000 no browser.

4

Abra o Claude Code

Em outro terminal (mantenha o preview rodando), abra o Claude Code dentro do projeto:

cd meu-video
claude

O Claude ja vai reconhecer o projeto Remotion e os Agent Skills instalados.

5

Descreva seu video

Agora e so descrever o que voce quer. O Claude vai criar os componentes React, registrar as compositions, e aplicar animacoes:

"Crie um video de 15 segundos com fundo branco.
Titulo 'Remotion e incrivel' aparece com fade-in no centro.
Apos 3 segundos, o titulo sobe e 3 bullet points
aparecem um por um de baixo para cima."
6

Veja o resultado no preview

Volte ao browser com o Remotion Studio aberto. O video ja vai estar la com hot reload. Use o player para navegar frame a frame e ajustar o que precisar.

Entendendo o codigo

Anatomia de um video Remotion

Todo video Remotion e composto por componentes React + metadata. Aqui esta o minimo necessario:

Componente do video

// src/MeuVideo.tsx
import { AbsoluteFill, useCurrentFrame, interpolate } from "remotion";

export const MeuVideo = () => {
  const frame = useCurrentFrame();

  // Opacidade vai de 0 a 1 nos primeiros 30 frames
  const opacity = interpolate(frame, [0, 30], [0, 1]);

  return (
    <AbsoluteFill style={{ background: "white" }}>
      <h1 style={{ opacity, textAlign: "center", marginTop: "40%" }}>
        Meu primeiro video!
      </h1>
    </AbsoluteFill>
  );
};

Registrar a Composition

// src/Root.tsx
import { Composition } from "remotion";
import { MeuVideo } from "./MeuVideo";

export const RemotionRoot = () => {
  return (
    <Composition
      id="MeuVideo"
      component={MeuVideo}
      durationInFrames={150}  // 5 seg a 30fps
      fps={30}
      width={1920}
      height={1080}
    />
  );
};
✓

Voce nao precisa escrever esse codigo manualmente. O Claude Code gera tudo isso quando voce descreve o video. Mas entender a estrutura ajuda a pedir ajustes mais precisos.

Exportar

Renderizar o video final

Quando estiver satisfeito com o preview, exporte para MP4:

# Render basico
npx remotion render MeuVideo out/video.mp4

# Com props dinamicas
npx remotion render --props='{"titulo": "Custom"}' MeuVideo

# Apenas um trecho (frames 0 a 90)
npx remotion render --frames=0-90 MeuVideo

# Escalar resolucao (720p para 1080p)
npx remotion render --scale=1.5 MeuVideo

# Formato vertical (Reels/Shorts)
# Defina width=1080 height=1920 na Composition
i

Para Reels e Shorts, use resolucao 1080x1920 (vertical) com fps=30. O Remotion suporta qualquer resolucao e aspect ratio.

Pro tips

Dicas e boas praticas

1

Itere no preview

Nao tente acertar tudo no primeiro prompt. Gere, veja o preview, e peca ajustes ao Claude.

2

Use interpolate()

A funcao interpolate() do Remotion e sua melhor amiga para animacoes suaves baseadas em frame.

3

Cenas separadas

Peca ao Claude para criar cada cena como componente separado. Facilita edicao e reuso.

4

Tailwind ajuda muito

Ative TailwindCSS no setup. O Claude gera estilos mais precisos e consistentes com utility classes.

✓

Dica de ouro: Seu video e codigo. Isso significa que voce pode versionar com Git, gerar variacoes automaticas, e escalar producao de conteudo com templates reutilizaveis.
